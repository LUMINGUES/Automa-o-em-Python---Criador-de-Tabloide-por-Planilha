# Automa-o-em-Python---Criador-de-Tabloide-por-Planilha
OFERTEIRO é uma poderosa ferramenta de automação desenvolvida em Python para transformar dados brutos de planilhas (dados.xlsx) em um Tabloide de Ofertas de alta qualidade no formato PDF. Utilizando as bibliotecas pandas e python-docx, o script gera um documento único, garantindo um layout padronizado de 16 produtos por página.
